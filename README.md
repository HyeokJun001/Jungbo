# 정처기 실기 이론 암기 트레이너

정보처리기사 실기 **이론 파트**를 주관식(단답형)으로 연습하는 웹앱. 설명을 읽고 용어를 직접 써 보며, 시험장처럼 손으로 답을 떠올리는 훈련에 초점을 맞췄습니다.

빌드·설치 없이 `index.html`을 브라우저로 열면 바로 실행됩니다. 순수 프론트엔드(HTML/CSS/바닐라 JS)이며, 진행 기록은 브라우저 `localStorage`에 저장됩니다.

## 실행

`index.html` 더블클릭. (별도 서버·인터넷 불필요)

## 기능

- **랜덤 / 섹터별 문제 풀기** — 350+ 문제, 17개 섹터
- **주관식 자동 채점** — 정답 변형·영문/한글·띄어쓰기 허용
- **💡 힌트 / 🔴 중점 키워드** — 토글로 켜고 끄기, 정답이 지문에 노출되면 자동 표시
- **📖 용어 설명 + 영문 풀네임** — 약어(UDDI, SAAM 등)는 풀네임과 설명을 함께 표시
- **🧩 빈칸 채우기** — 이론 문장의 핵심어 채우기
- **📚 섹터별 이론 공부** — 요약 정리 + 핵심 키워드 자동 빨강 강조 + 두문자 암기팁
- **🔍 검색** — 용어·설명·섹터로 문제 찾기
- **📋 기록·복기** — 맞은/틀린 문제를 한 곳에서, 내 답·정답·설명까지 복습
- **⭐ 북마크** — 별표한 문제 모음
- **✍️ 틀리면 3번 쓰기 (선택)** — 틀린 문제의 "정답 : 설명" 문장을 3번 정확히 써야 다음으로 진행. 한컴타자연습식 실시간 색칠(맞으면 초록/틀리면 빨강), `·`는 쉼표와 동일 취급
- **⏹ 그만하기** — 세션 도중 지금까지 푼 결과를 한눈에 요약

## 파일 구조

| 파일 | 내용 |
|------|------|
| `index.html` | 진입점 |
| `app.js` | 앱 로직 (라우팅·채점·기록·벌칙 등) |
| `questions.js` | 문제 은행 (`QUESTIONS`, `SECTORS`) |
| `theory.js` | 섹터별 이론 요약 + 빈칸 문제 |
| `glossary.js` | 용어 사전 (약어 풀네임·설명) |
| `styles.css` | 스타일 |

문제 추가는 `questions.js`의 `QUESTIONS` 배열에 객체 하나를 넣으면 됩니다.
